### 享元模式（Flyweight Pattern）：属于结构型模式

------

[TOC]

##### 00：概述

- 用于减少创建对象的数量，以减少内存占用和提高性能，复用现有的同类对象，如果未找到匹配的对象，则创建新对象；

###### 实现

- 用唯一标识码判断，如果在内存中有，则返回这个唯一标识码所标识的对象；
- 用 HashMap 存储这些对象；

###### 示例：

- JAVA 中的 String 的字符串常量池，如果有则返回，如果没有则创建一个字符串保存在字符串缓存池里面；

##### 01：示例

```Java
public native String intern();
```
