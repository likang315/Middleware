##### 01：MyISAM 和 InnoDB 的区别

| MyISAM                                                       | InnoDB                                                       |      |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ---- |
| MySQL 5.5 之前默认的存储引擎；                               | MySQL 5.5 之后默认的存储引擎；                               |      |
| 不支持事务，表锁设计，但是支持全文索引；                     | 事务型存储引擎，支持回滚、崩溃恢复能力； 支持行锁，表锁设计，支持外键； |      |
| 不支持外键；                                                 | 支持外键；                                                   |      |
| MyISAM 存储引擎的索引和数据是分开存储的，索引文件和数据文件分别存储； | 数据和索引在一个表空间中；                                   |      |
