##### 01：如何查询出每个班级语文成绩前三名的同学记录？

- ```sql
  # 关联子查询：为每一个候选行执行一次
  SELECT
      a.`id`,
      a.class,
      a.`name`,
      a.chinese,
      count(b.`id`) + 1 as rank_num
  from scores a
  left join scores b on a.class = b.class
      and a.chinese < b.chinese
  group by a.`id`,
      a.class,
      a.`name`,
      a.chinese
  HAVING rank_num <=3
  order by a.class asc,a.chinese desc
  ```
