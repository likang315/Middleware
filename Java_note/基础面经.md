1：Java有没有goto语句？
	其中有 goto 和 const，但是这两个是目前无法使用的关键字，因此有些地方将其称之为保留字
2：在Java中，如何跳出当前的多重嵌套循环
	在最外层循环前加一个标记如 A，然后用 break A;
3：当一个对象被当作参数传递到一个方法后，到底是值传递还是引用传递？
	是值传递，对象传递的是副本，这个副本存放的是对象的地址（引用），所以修改对象的属性时，会影响到原对象，但是String时一个例外，因为它是fianl的，会创建新的对象。不管传递的值是什么数据类型，都是值传递
4： 抽象的(abstract)方法是否可同时是静态的(static), 是否可同时是本地方法 (native)，是否可同时被synchronized
	都不可以，static是静态的，不可以被重写，native本地代码（ C 语言）实现的方法，而不能被重写，synchronized与方法的实现细节有关不可以被重写
5：final、finally、finalize的区别？ 
	final：用于声明属性，方法和类，分别表示属性不可变，方法不可覆盖，被其修饰的类不可继承。
	finally：异常处理语句结构的一部分，表示总是执行
	finalize：Object类的一个方法，在垃圾回收器执行的时候会调用被回收对象的此方法
6：short s1 = 1; s1 = s1 + 1; 有错吗?short s1 = 1; s1 += 1 有错吗
	需要强制类型转换，第二个隐含有强制类型转换
7：匿名内部类不能访问外部类方法中的局部变量，除非该变量被声明为final类型
	编译器实现上的困难：内部类对象的生命周期很有可能会超过局部变量的生命周期，声明成final，编译器会把匿名内部类对象要访问的所有final类型局部变量，都拷贝一份作为自己的成员变量
7：System.out.println()的真正含义吗？
	System是java.lang里面的一个类，out就是System里面的一个静态属性成员，并且它是java.io.PrintStream类的引用
	println()就是java.io.PrintStream类里的一个方法，它的作用是向控制台输出信息
